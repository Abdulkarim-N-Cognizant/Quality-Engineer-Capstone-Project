API TEST CASES

ID: TC_API_01
Title: GET /posts Returns 200 OK
Testing Level: Integration
Test Type: Functional – Positive
Preconditions: API is reachable
Steps:
1. Send GET request to https://jsonplaceholder.typicode.com/posts
Expected Result:
- Response code 200 OK
- Response body contains an array of posts
- Posts contains all required fields
Actual Result: A GET request was sent to https://jsonplaceholder.typicode.com/posts.
The API returned a 200 OK response. The response body contained an array of posts,
and each post included the required fields: id, userId, title, and body.
Status: TEST PASS
